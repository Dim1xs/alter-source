Lua Interface Documentation

Functions

Server Functions

luaMsg(message)

Prints a message to the game console.

Parameters: message - The message to be printed.

luaConMsg(message)

Prints a message to the console with timestamp.

Parameters: message - The message to be printed.

luaGetTime()

Gets the current server time.

Returns: The current server time.

luaGetCVar(cvarName)

Gets the value of a console variable.

Parameters: cvarName - The name of the console variable.

Returns: The value of the specified console variable.

luaSetCVar(cvarName, value)

Sets the value of a console variable.

Parameters: cvarName - The name of the console variable.
value - The value to set.

luaExecuteConsoleCommand(command)

Executes a console command on the server.

Parameters: command - The console command to execute.

luaSay(message1, message2, ...)

Sends a message to all players.

Parameters: message1, message2, ... - Messages to be sent. Multiple messages can be provided.

luaLoadFile(filePath)

Loads and executes a Lua script file.

Parameters: filePath - The path to the Lua script file.

Player Functions

luaGetPlayerName(playerIndex)

Gets the name of the player with the specified index.

Parameters: playerIndex - The index of the player.

Returns: The name of the player.

luaGetPlayerHealth(playerIndex)

Gets the health of the player with the specified index.

Parameters: playerIndex - The index of the player.

Returns: The health of the player.

luaGetPlayerPosition(playerIndex)

Gets the position of the player with the specified index.

Parameters: playerIndex - The index of the player.

Returns: Three values representing x, y, and z coordinates of the player's position.

luaHandlePlayerInput(playerIndex, input)

Handles player input, such as performing actions like jumping.

Parameters: playerIndex - The index of the player.
input - The input to handle (e.g., "jump").

Returns: True if the player exists, false otherwise.

luaGetPlayerCount()

Gets the total number of players in the game.

Returns: The total number of players.

luaGetPlayerTeam(playerIndex)

Gets the team number of the player with the specified index.

Parameters: playerIndex - The index of the player.

Returns: The team number of the player.

luaGetPlayerMaxSpeed(playerIndex)

Gets the maximum speed of the player with the specified index.

Parameters: playerIndex - The index of the player.

Returns: The maximum speed of the player.

luaGiveItem(playerIndex, itemName)

Gives an item to the player with the specified index.

Parameters: playerIndex - The index of the player.
itemName - The name of the item to give.

luaIsPlayerValid(playerIndex)

Checks if the player with the specified index is valid.

Parameters: playerIndex - The index of the player.

Returns: True if the player is valid, false otherwise.

luaGetPlayerArmor(playerIndex)

Gets the armor value of the player with the specified index.

Parameters: playerIndex - The index of the player.

Returns: The armor value of the player.

luaIsPlayerAlive(playerIndex)

Checks if the player with the specified index is alive.

Parameters: playerIndex - The index of the player.

Returns: True if the player is alive, false otherwise.

luaKillPlayer(playerIndex)

Kills the player with the specified index.

Parameters: playerIndex - The index of the player.

luaSetPlayerArmor(playerIndex, armorValue)

Sets the armor value of the player with the specified index.

Parameters: playerIndex - The index of the player.
armorValue - The armor value to set.

Entity Functions

luaGetEntityPosition(entityIndex)

Gets the position of the entity with the specified index.

Parameters: entityIndex - The index of the entity.

Returns: Three values representing x, y, and z coordinates of the entity's position.

luaGetEntityClassname(entityIndex)

Gets the classname of the entity with the specified index.

Parameters: entityIndex - The index of the entity.

Returns: The classname of the entity.

luaSpawnEntity(classname, x, y, z)

Spawns an entity with the specified classname and position.

Parameters: classname - The classname of the entity to spawn.
x, y, z - The coordinates of the entity's position.

luaTeleportPlayer(playerIndex, x, y, z)

Teleports the player with the specified index to the given position.

Parameters: playerIndex - The index of the player to teleport.
x, y, z - The coordinates of the destination position.

luaSetEntityProperty(entityIndex, property, value)

Sets a property of the entity with the specified index.

Parameters: entityIndex - The index of the entity.
property - The property to set.
value - The value to set the property to.

luaIsPlayerNearEntity(playerIndex, entityIndex, radius)

Checks if the player with the specified index is near the entity with the specified index within the given radius.

Parameters: playerIndex - The index of the player.
entityIndex - The index of the entity.
radius - The radius to check within.

Returns: True if the player is near the entity, false otherwise.

luaGetEntityVelocity(entityIndex)

Gets the velocity of the entity with the specified index.

Parameters: entityIndex - The index of the entity.

Returns: Three values representing the x, y, and z components of the entity's velocity.

luaKillEntity(entityIndex)

Destroys the entity with the specified index.

Parameters: entityIndex - The index of the entity to destroy.

luaSetPlayerMaxSpeed(playerIndex, speed)

Sets the maximum speed of the player with the specified index.

Parameters: playerIndex - The index of the player.
speed - The maximum speed to set.

luaGetEntityName(entityIndex)

Gets the name of the entity with the specified index.

Parameters: entityIndex - The index of the entity.

Returns: The name of the entity.

luaGetEntityModel(entityIndex)

Gets the model name of the entity with the specified index.

Parameters: entityIndex - The index of the entity.

Returns: The model name of the entity.

luaGetEntityClassId(entityIndex)

Gets the class ID of the entity with the specified index.

Parameters: entityIndex - The index of the entity.

Returns: The class ID of the entity.
