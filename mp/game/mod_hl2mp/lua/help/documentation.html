Lua Script Interface Documentation

Concommands

reload_lua: Command to reload Lua scripts.

Global Constants

  • FOR_ALL_PLAYERS: -1
  • INVALID_ENTITY: -1
  • NULL: 0
  • MAX_HEALTH: 100
  • MAX_ARMOR: 200
  • TEAM_ONE: Combines
  • TEAM_TWO: Rebels
-- Example in Lua:
Msg(FOR_ALL_PLAYERS) -- Output: -1
Msg(TEAM_ONE) -- Output: Combines

Global Variables

  • GAMEMODE: Value of the "as_gamemode" cvar
  • TEAMPLAY: Value of the "mp_teamplay" cvar
  • COOP: Value of the "coop" cvar
  • WEAPON_MELEE_SLOT: Melee weapon slot
  • WEAPON_SECONDARY_SLOT: Secondary weapon slot
  • WEAPON_PRIMARY_SLOT: Primary weapon slot
  • WEAPON_EXPLOSIVE_SLOT: Explosive weapon slot
  • WEAPON_TOOL_SLOT: Tool weapon slot
  • MAX_FOV: Maximum field of view
  • MAX_PLAYERS: Maximum number of players
  • TEAM_NONE: Team none
  • TEAM_SPECTATOR: Team spectator
  • TEAM_ANY: Team any
  • TEAM_INVALID: Team invalid
  • HUD_PRINTNOTIFY: Print notify to HUD
  • HUD_PRINTCONSOLE: Print to console
  • HUD_PRINTTALK: Print talk to HUD
  • HUD_PRINTCENTER: Print center to HUD
-- Example in Lua:
Msg(GAMEMODE) -- Output: "Sandbox"
Msg(MAX_PLAYERS) -- Output: 32

Lua Functions

  • GetTime: Get the current time
  • GetCVar: Get the value of a ConVar
  • ExecuteConsoleCommand: Execute a console command
  • Say: Send a message to all players
-- Example in Lua:
local currentTime = GetTime()
Msg("Current server time: " .. currentTime)

local coopMode = GetCVar("coop")
Msg("Coop is " .. coopMode)

ExecuteConsoleCommand("mp_restartgame 1")

Say("Hello, players!")

Player Functions

  • GetPlayerName: Get the name of a player by index
  • GetPlayerHealth: Get the health of a player by index
  • HandlePlayerInput: Handle player input
  • GetPlayerPosition: Get the position of a player by index
  • GetPlayerCount: Get the number of players
  • GetPlayerTeam: Get the team of a player by index
  • GetPlayerMaxSpeed: Get the max speed of a player by index
  • GiveItem: Give an item to a player
  • KillPlayer: Kills a player
-- Example in Lua:
local playerName = GetPlayerName(1)
Msg("Player 1's name is " .. playerName)

local playerHealth = GetPlayerHealth(1)
Msg("Player 1's health is " .. playerHealth)

local success = HandlePlayerInput(1, "jump")
Msg("Player 1 jumped: " .. tostring(success))

local x, y, z = GetPlayerPosition(1)
Msg("Player 1's position is (" .. x .. ", " .. y .. ", " .. z .. ")")

local playerCount = GetPlayerCount()
Msg("Number of players: " .. playerCount)

local playerTeam = GetPlayerTeam(1)
Msg("Player 1's team is " .. playerTeam)

local playerMaxSpeed = GetPlayerMaxSpeed(1)
Msg("Player 1's max speed is " .. playerMaxSpeed)

GiveItem(1, "weapon_crowbar")
Msg("Player 1 received a crowbar")

KillPlayer(1)
Msg("POW! You are dead! Not big surprise.")

Entity Functions

  • GetEntityPosition: Get the position of an entity by index
  • SpawnEntity: Spawn an entity
-- Example in Lua:
local x, y, z = GetEntityPosition(1)
Msg("Entity 1's position is (" .. x .. ", " .. y .. ", " .. z .. ")")

-- Example to spawn an entity (assuming SpawnEntity function is implemented)
SpawnEntity("npc_citizen", x, y, z)
Msg("Spawned npc_citizen at (" .. x .. ", " .. y .. ", " .. z .. ")")

Server Functions

  • ExecuteConsoleCommand: Execute a console command on the server
  • SetCVar: Set the value of a ConVar
  • Msg: Print a message to console
  • ConMsg: Print a message to console and log
  • LoadFile: Loads a Lua file.
-- Example in Lua:
ExecuteConsoleCommand("sv_cheats 1")

SetCVar("mp_friendlyfire", "1")

Msg("This is a message to the console")

ConMsg("This is a message to the console and log")

LoadFile("lua/script.lua")
